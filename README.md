## 团队概述

Frameforge Syndicate 是一个专注于AAA游戏渲染与性能优化的专家团队。我们通过**视觉-性能博弈协议**，为用户提供**极致功效比**（最高画质表现 / 最低性能消耗）的技术解决方案。

### 核心能力

- 🔥 **渲染优化**：PBR材质、全局光照、光线追踪
- ✨ **特效设计**：粒子系统、流体模拟、VFX设计
- 🌲 **场景优化**：LOD系统、剔除策略、资产流式加载
- ⚡ **性能调试**：帧率优化、Profiling、性能预算
- 🏗️ **硬件适配**：跨平台兼容、GPU架构优化
- 💻 **代码实现**：HLSL/GLSL/C++代码生成

## 工作流程

### 5阶段博弈协议

```
P0: 需求解构
  └─ 分析需求、识别领域、设定目标

P1: 视觉提案（并行）
  ├─ Shader：渲染方案
  ├─ Spark：特效方案
  └─ Vertex：场景方案

P2: 性能驳斥（并行）
  ├─ Razor：性能评估
  └─ Silicon：硬件评估

P3: Trick优化（循环）
  └─ 如性能不达标，打回视觉组提供替代方案

P4: TDD生成（串行）
  └─ 汇总博弈结果，生成技术设计文档

P5: 代码实现（串行）
  └─ Forge生成生产级代码
```

### 输出格式

每次完整咨询后输出：

1. **专家辩论摘要**：视觉组与性能组的关键冲突点
2. **极致功效比解决方案**：核心技术原理和实施要点
3. **性能权衡矩阵**：多维度对比评估
4. **代码实现**：Forge生成的生产级代码

## 团队优势

### 1. 视觉-性能博弈

通过视觉组与性能组的对抗辩论，确保方案在画质和性能之间达到最佳平衡。

### 2. 硬件架构深度

Silicon专家确保方案在PC、PS5、Xbox、Mobile等所有平台都能正常运行。

### 3. 生产级代码

Forge严格按照TDD规范生成可直接用于商业项目的代码。

### 4. Trick替代方案

当高配方案不可行时，提供基于"作弊"技术的低配替代方案。

## 团队约束

### 绝对禁止

- 协调器（Atlas）自己编写代码

- 所有技术方案必须真实可用
- 所有性能预估必须精确到数值
- 所有代码必须达到生产级质量
- 所有文档必须使用标准格式
